## 🎨 주요 기능

### 📊 1. 인포그래픽 생성

**용도**: 복잡한 정보를 시각적으로 표현

**사용 방법**:
1. "설명" 입력 필드에 생성하고 싶은 인포그래픽 설명 입력
   - 예: "2026년 한국 교육 시장 현황"
   - 예: "AI 시대 교육 혁신 전략"

2. "생성 라운드" 선택
   - 1: 빠른 생성
   - 2-3: 품질 향상 (시간 더 소요)

3. "출력 모드" 선택
   - 친화적: 간단한 설명
   - 상세: 자세한 분석

4. **생성하기** 클릭

**결과**:
- Task ID 발급 (진행 상황 추적용)
- 인포그래픽 생성 대기열 추가

---

### 📑 2. PPT 생성

**용도**: 프레젠테이션 자동 생성

**사용 방법**:
1. **제목** 입력
   - 예: "2024 교육 기술 보고서"

2. **내용** 입력 (마크다운 형식)
   ```markdown
   # 개요
   교육 시장 현황 분석
   
   ## 주요 내용
   1. 시장 규모
   2. 기술 트렌드
   3. 경쟁 구도
   ```

3. **모드** 선택
   - 표준: 텍스트 중심, 읽기 쉬움
   - 창작: 이미지 중심, 시각적 임팩트

4. **슬라이드 수** 지정 (3-50장)

5. **생성하기** 클릭

**팁**:
- 마크다운 형식 지원 (# 제목, ## 부제목 등)
- Ctrl+Enter로 빠르게 생성 가능

---

### 📈 3. 데이터 분석

**용도**: Excel, CSV 파일 분석 및 시각화

**지원 형식**:
- .xlsx (Excel)
- .csv (CSV)
- .xls (구 Excel)

**사용 방법**:
1. **파일 업로드** 방식 선택
   - 파일 드래그 & 드롭
   - 또는 클릭해서 선택

2. **분석 유형** 선택
   - 요약: 기본 통계
   - 상세 분석: 깊이 있는 분석
   - 내보내기: 처리된 데이터 다운로드

3. **분석하기** 클릭

**지원 기능**:
- 자동 데이터 정제
- 통계 분석
- 시각화
- 대용량 파일 처리 (10,000+ 행)

---

### 🔬 4. 심층 연구

**용도**: 주제에 대한 다각적 분석 및 보고서 생성

**사용 방법**:
1. **연구 주제** 입력
   - 예: "한국 교육 기술 시장"
   - 예: "AI 시대 교사의 역할 변화"

2. **연구 차원** 입력 (각 줄에 하나씩)
   ```
   시장 규모
   기술 트렌드
   경쟁 구도
   규제 및 정책
   ```
   최소 2개, 최대 10개 권장

3. **깊이 수준** 선택
   - 간단: 빠른 개요
   - 표준: 균형잡힌 분석 ⭐ 추천
   - 심화: 상세한 분석 (시간 오래 걸림)

4. **연구 시작** 클릭

**연구 프로세스**:
- 📋 연구 계획 수립
- 🔍 차원별 증거 수집
- 🧩 증거 종합 및 분석
- 📝 최종 보고서 작성

## 📊 작업 현황 추적

### 최근 작업 목록
페이지 하단의 "최근 작업 현황" 섹션에서:
- 실행한 모든 작업 확인
- Task ID로 진행 상황 추적
- 작업 상태 (대기 중, 완료, 오류)
- 작업 실행 시간

### Task ID 활용
```
형식: YYYYMMDD_HHMMSS_XXXXXXXX
예: 20260505_213900_780e556c

API에서 상태 확인:
GET /api/v1/status/{task_id}
```

## 💡 사용 팁

### 1. 효율적인 인포그래픽 생성
- **구체적인 설명**: "AI 교육" ❌ → "AI 시대 교사 역량 강화 방안" ✅
- **여러 라운드**: 품질 향상 원하면 2-3 라운드 선택
- **키워드 활용**: 핵심 키워드 포함하면 더 정확한 결과

### 2. PPT 작성 요령
- **구조화된 콘텐츠**: 제목→개요→본론→결론 순서
- **마크다운 활용**: 헤더(#)로 계층 구조 명확히
- **이미지 중심**: 창작 모드에서 시각적 임팩트
- **텍스트 중심**: 표준 모드에서 읽기 편함

### 3. 데이터 분석 팁
- **정제된 데이터**: 헤더 행 포함, 빈 행/열 제거
- **대용량 파일**: 자동 Parquet 변환 (성능 향상)
- **분석 유형**: 상세 분석이 더 깊이 있는 인사이트 제공

### 4. 심층 연구 최적화
- **차원 설계**: 4-6개 차원이 가장 균형잡힘
- **깊이 선택**: 표준이 시간 대비 효율 최고
- **구체적 주제**: 광범위한 주제보다 특정 주제가 더 정확

## 🐛 문제 해결

### 1. "API 서버에 연결할 수 없습니다"
**원인**: 서버가 실행되지 않음
**해결**:
```bash
source sensenova_env/bin/activate
python3 sensenova_api_server.py 8001
```

### 2. 파일 업로드 실패
**원인**: 파일 형식 또는 크기
**해결**:
- 지원 형식 확인: .xlsx, .csv, .xls
- 파일 크기: 100MB 이하
- 파일명 특수문자 제거

### 3. 느린 생성 속도
**원인**: 서버 과부하 또는 복잡한 요청
**해결**:
- 인포그래픽: 라운드 수 줄임
- PPT: 슬라이드 수 줄임
- 다른 작업이 완료될 때까지 대기

### 4. 브라우저 오류
**원인**: 캐시 또는 JavaScript 오류
**해결**:
- 브라우저 캐시 초기화 (Ctrl+Shift+Delete)
- 개발자 도구 열기 (F12) → 콘솔에서 오류 확인
- 다른 브라우저로 시도
